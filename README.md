# Amazon-Sales-Data-Analysis

🔍 Project Overview

This project presents an interactive Amazon Sales Dashboard built using Power BI to analyze sales performance, customer preferences, and operational efficiency.
The dashboard provides insights into product categories, order fulfillment, delivery performance, customer size preferences, and regional demand, enabling data-driven decision-making.

🎯 Objectives
Analyze overall sales performance
Identify top-performing product categories
Understand customer purchase patterns (size, region)
Evaluate order fulfillment and delivery efficiency
Detect data gaps and operational inefficiencies
